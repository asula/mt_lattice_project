Folder input (est) cointains baseline input and three lattice files obtained with different configurations. Folder output (eng) contains the translations obtained with corresponding input files. Folder target (eng) contains the target translation.

INPUT (EST)

input_1k_baseline.et - baseline input
input_1k_lattice1.plf - lattice file generated based on the original text
input_1k_lattice2.plf - lattice file generated based on lemmatized text
input_1k_lattice3.plf - lattice file generated based on both the original text and lemmatized text


OUTPUT (ENG)

output_1k_baseline.en - translation obtained with input_1k_baseline.et as input
output_1k_lattice1.en - translation obtained with input_1k_lattice1.plf as input
output_1k_lattice2.en - translation obtained with input_1k_lattice2.plf as input
output_1k_lattice3.en - translation obtained with input_1k_lattice2.plf as input


TARGET (ENG)

target_1k.en - original output translation
